# 🚀 UNV-HMND — AI Modular System

AI-powered system built with clean architecture principles.

---

## 🧠 Project Overview

UNV-HMND is a modular AI system designed to simulate real-world backend architecture with:

- Planner
- Executor
- Validator

It connects a frontend interface with a backend system capable of processing AI-driven tasks.

---

## ⚙️ Tech Stack

- TypeScript
- Node.js
- Vite
- Clean Architecture
- Modular Design
