# Non Goals

## Purpose

This document explains what the project does not aim to become.

Its purpose is to preserve clarity and credibility.

## The Project Is Not

- AGI;
- artificial consciousness;
- a digital soul;
- omniscience;
- mind control;
- human scoring;
- surveillance system;
- autonomous operator;
- guaranteed success machine.

## The Project Does Not Replace

- human responsibility;
- learning;
- judgment;
- relationships;
- effort;
- community.

## The Project Does Not Promise

- guaranteed outcomes;
- automatic wealth;
- perfect decisions;
- universal solutions.

## Core Rule

Proposal != Execution.

Approve != Execute.

Human approval remains mandatory.

## Core Pattern

Understand limits

↓

Protect coherence

↓

Build responsibly
