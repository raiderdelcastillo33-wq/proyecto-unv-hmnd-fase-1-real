# RAIDER HOLBERTON OS Operating Model

## 1. ChatGPT as Brain and Mentor

ChatGPT acts as the brain and mentor.

Its role is to:

- explain;
- organize ideas;
- maintain coherence;
- review;
- teach;
- guide decisions.

ChatGPT does not replace human responsibility.

## 2. Notion AI as Secretary and Operating System

Notion AI acts as secretary and operating system.

Its role is to:

- organize information;
- summarize;
- classify;
- prepare context;
- detect pending items.

It does not invent work.

It does not replace the mentor.

## 3. Codex as Controlled Builder

Codex acts as a controlled builder.

Its role is to:

- implement approved changes;
- generate code;
- improve documentation;
- perform controlled modifications.

Codex does not decide priorities.

Codex does not bypass approval.

## 4. Holberton as Foundation

Holberton provides the fundamentals:

- Linux;
- Shell;
- Git;
- GitHub;
- C;
- Python;
- Docker;
- communication;
- teamwork.

Fundamentals come before complexity.

## 5. Git and GitHub as Delivery System

Git and GitHub provide:

- version control;
- history;
- verification;
- collaboration;
- delivery.

Every change should end with a clean repository.

- do not invent;
