# System Relationship Map

## 1. Purpose

This document explains how the main systems work together:

* RAIDER HOLBERTON OS
* HUMAN INTERVIEW OS
* UNV-HMND
* Humanity Pro University

## 2. System Roles

RAIDER HOLBERTON OS is the learning and execution system.

HUMAN INTERVIEW OS is the people, interviews, problems, plans, follow-up, and review system.

UNV-HMND is the technical architecture and construction system.

Humanity Pro University is the human growth and autonomy vision.
